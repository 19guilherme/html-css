Harmonia das Cores

Simetria das cores é mto importante, e a sicronia é uma arma poderosa para o designer do site.

SEMPRE AO FAZER UM SITE, ANALISAR AS CORES CONFORME A NECESSIDADE DO SITE. CADA COR TEM SUAS EMOÇÕES E SENTIDO.

Cores Primárias

vermelho, azul e amarelo

Cores Secundárias

verde, roxo e laranja

Amarelo + vermelho = laranja
Vermelho + azul = roxo ou violeta
Azul + amarelo = verde

Cores terciárias

Chamado sempre pelo 1º nome Primário, depois pelo 2º secundário.

amarelo-alaranjado,vermelho-alaranjado, vermelho-arroxeado, azul-arroxeado, azul-esverdeado, amarelo-esverdeado

Amarelo + laranja = amarelo-alaranjado (similar ao bege)
Vermelho + laranja = vermelho-alaranjado ou laranja-escuro
Vermelho + roxo = vermelho-arroxeado (similar ao vinho)
Azul + roxo = azul-arroxeado
Amarelo + verde = amarelo-esverdeado ou verde-claro
Azul + verde = azul-esverdeado (similar ao verde-água)

Obs: * Recomenda-se escolher entre 3 à 5 cores para palhetas de cores do site.
